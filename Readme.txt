Para carregar o pacman no eclipse:

File -> New -> Java Project
  unckeck: use default location
  Browse- selecionar a raiz do diretório onde o pacman está instalado.
  Finish


Para correr, ir para a class principal:
	pacman\src\screenpac.model.Game 
e correr.

Implementação do algoritmo pode ser encontrada em src/screencpac.controllers.IA2122PL1G4
